# VRML-expander（CLIツール）

## 概要
3Dモデル記述言語「VRML 2.0」形式のファイルにおいて、外部参照を行う「Inlineノード」を自動検出し、参照先のファイルを展開・統合して単一のファイルにまとめるCLIツールです。複数ファイルに分割されたシーンの管理や共有を効率化するために開発しました。

## 技術的工夫

- **構文解析の実装：** 高精度なノード検知を実現するため、独自のトークナイザを実装し、字句解析を通じてInlineノードを特定する仕組みを構築しました。
- **信頼性と品質の確保：** 循環参照（ファイル同士が互いを参照し合うケース）を検知し、適切なエラーハンドリングを行う機能を実装することで、システムの異常終了を防止しました。また、トークナイズやファイル展開処理に対してユニットテストや全体テストを導入し、動作の保証をしました。
- **最適化と制約の明示：** 統合後のデータサイズを少なくするために、書き出し内容の最適化を行いました。
- **仕様の明確化：**公式ドキュメントを精読して仕様を整理し、ツールが対応可能な文法制約を明文化することで、ユーザーが安心して利用できる設計としました。
