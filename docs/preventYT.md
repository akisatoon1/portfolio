# YouTube タイムスタンプ・スクロール防止拡張機能 (prevent-timestamp)

## 概要
YouTubeのコメント欄や概要欄にあるタイムスタンプ（「10:35」などのリンク）をクリックした際、ページ最上部へ強制的にスクロールされる仕様を防止し、視聴位置を維持するブラウザ拡張機能です。画面分割でコメントを表示しながら視聴する際の利便性を向上させます。

## 技術的工夫と取り組み

- **仕様変更への耐性向上：** 当初はCSSセレクタを用いてHTML要素を特定していましたが、YouTubeのUI変更の影響を受けやすいという課題がありました。そこで、正規表現を用いてテキストパターンから直接タイムスタンプを判定するアルゴリズムに変更し、保守性と堅牢性を高めました。
- **適用範囲の拡大：** コメント欄だけでなく、概要欄のタイムスタンプにも対応させました。
- **SPAへの対応：** YouTubeのサイト構造がSPAであることを考慮し、動画視聴ページ（/watch）以外から遷移した場合でも、拡張機能が適切に読み込まれ、常に機能が維持されるよう実装を工夫しました。
