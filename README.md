## 見てほしいもの
### Youtubeを便利にする拡張機能
[詳しい解説はこちら](https://github.com/akisatoon1/portfolio/blob/main/docs/preventYT.md)
#### 概要
YouTubeのコメント欄や概要欄にあるタイムスタンプ（「10:35」などのリンク）をクリックした際、ページ最上部へ強制的にスクロールされる仕様を防止し、視聴位置を維持するブラウザ拡張機能です。画面分割でコメントを表示しながら視聴する際の利便性を向上させます。

### 点群データから東京を3Dモデルで再現
[詳しい解説はこちら](https://github.com/akisatoon1/portfolio/blob/main/docs/3dmodel.md)
#### 概要
1m間隔の格子状標高値（電子地図データ）を解析し、東京ドーム周辺の都市街区を3Dモデルで再現するプロジェクトです。欠損値（標高値 -9999.99）を含む膨大な点群データから、建物と地表を判別して可視化しました。

### 3Dモデル言語の複数ファイルをひとつに統合
[詳しい解説はこちら](https://github.com/akisatoon1/portfolio/blob/main/docs/vrml.md)
#### 概要
3Dモデル記述言語「VRML 2.0」形式のファイルにおいて、外部参照を行う「Inlineノード」を自動検出し、参照先のファイルを展開・統合して単一のファイルにまとめるCLIツールです。複数ファイルに分割されたシーンの管理や共有を効率化するために開発しました。
